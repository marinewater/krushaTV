success
directive in module krushaTV

Description

displays an symbol for a specified amount of time to indicate if a action has failed or succeeded

Usage

as element:
<success
       timeout="{number}"
       success="{boolean}">
</success>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
timeoutnumber

timeout in milliseconds

successboolean

true for success, false for error
