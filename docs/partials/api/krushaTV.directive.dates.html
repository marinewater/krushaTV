dates
directive in module krushaTV

Description

helper directive for calendar directive

Usage

as attribute

     days="{Array}"
     change-to-day="{function}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
daysArray

changeToDayfunction
