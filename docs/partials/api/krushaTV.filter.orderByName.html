orderByName
filter in module krushaTV

Description

"orderByName" sorts an array by name, but does not take a leading "The" into account (e.g. "The Apple" would appear before "Bingo").

Usage

In HTML Template Binding

{{ orderByName_expression | orderByName[:attribute] }}

In JavaScript

$filter('orderByName')(shows[, attribute])

Parameters

ParamTypeDetails
showsArray

shows

attribute
(optional)
string

Name of the attribute which value should be used for sorting. Defaults to 'name'.
