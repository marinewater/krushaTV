copyEpisode
filter in module krushaTV

Description

"copyEpisode" creates a string that is later used to be copied to the clipboard. Looks like "Firefly S01E04".

Usage

In HTML Template Binding

{{ copyEpisode_expression | copyEpisode[:show][:season] }}

In JavaScript

$filter('copyEpisode')(episode[, show][, season])
