watched
directive in module krushaTV

Description

displays all watched/unwatched episodes and lets the user mark them as unwatched/watched

Usage

as element:
<watched
       shows="{object}"
       mark-episode-watched="{function}"
       mark-season-watched="{function}"
       mark-show-watched="{function}"
       watched="{boolean}">
</watched>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
showsobject

object containing watchted/unwatched episodes

markEpisodeWatchedfunction

function that marks an episode as watched/unwatched

markSeasonWatchedfunction

function that marks a season and all of its episodes as watched/unwatched

markShowWatchedfunction

function that marks a show and all of its seasons and episodes as watched/unwatched

watchedboolean

displays the correct words (watched/unwatched): true: shows unwatched episodes, false: shows watched episodes
