unwatched
directive in module krushaTV

Description

displays all watched/unwatched episodes and lets the user mark them as unwatched/watched

Usage

as element:
<unwatched
       shows="{object}"
       seasons="{object}"
       episodes="{object}">
</unwatched>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
showsobject

object containing shows with watched/unwatched episodes

seasonsobject

object containing seasons with watched/unwatched episodes

episodesobject

object containing watched/unwatched episodes
