unwatched
directive in module krushaTV

Description

displays all watched/unwatched episodes and lets the user mark them as unwatched/watched

Usage

as element:
<unwatched
       shows="{object}"
       seasons="{object}"
       episodes="{object}"
       get-seasons="{function}"
       get-episodes="{function}"
       mark-show-watched="{function}"
       mark-season-watched="{function}"
       mark-episode-watched="{function}"
       watched="{boolean}">
</unwatched>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
showsobject

object containing shows with watched/unwatched episodes

seasonsobject

object containing seasons with watched/unwatched episodes

episodesobject

object containing watched/unwatched episodes

getSeasonsfunction

function that receives all seasons for a specified show

getEpisodesfunction

function that receives all episodes for a specified season

markShowWatchedfunction

function that marks all episodes of a show as watchted/unwatched

markSeasonWatchedfunction

function that marks all episodes of a season as watchted/unwatched

markEpisodeWatchedfunction

function that marks a single episode as watchted/unwatched

watchedboolean

determined if this template shows watched or unwatched episodes
