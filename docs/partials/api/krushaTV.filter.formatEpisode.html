formatEpisode
filter in module krushaTV

Description

"formatEpisode" creates a string like "S02E03" (stands for Season 2 Episode 3)

Usage

In HTML Template Binding

{{ formatEpisode_expression | formatEpisode:season }}

In JavaScript

$filter('formatEpisode')(episode, season)

Parameters

ParamTypeDetails
episodenumber

episode number

seasonnumber

season number
