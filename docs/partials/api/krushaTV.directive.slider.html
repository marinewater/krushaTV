slider
directive in module krushaTV

Description

displays a slider, lets the user easly choose a value in a specified range

Usage

as element:
<slider
       min="{number}"
       max="{number}"
       step="{number}"
       ng-model="{expression}"
       on-slide-start="{function}"
       on-slide-stop="{function}">
</slider>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
minnumber

slider's floor

maxnumber

slider's ceiling

stepnumber

steps between slider's values

ngModelexpression

model

onSlideStartfunction

callback, called when the user grabs the slider's handle

onSlideStopfunction

callback, called when the user releases the slider's handle
